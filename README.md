# playground

## How to Run

1. Start the application using Docker Compose:
```bash
docker-compose up -d
```

2. Access the application:
- URL: http://localhost:8083
- Login credentials:
  - Username: admin
  - Password: admin

## Notes
- The application runs on port 8083
- Use the `-d` flag with docker-compose to run in detached mode (background)
